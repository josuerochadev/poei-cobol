# Cours - Formation Mainframe

Ce dossier contient les supports de cours organisés par module.

## Modules disponibles

| Module | Description | Chapitres |
|--------|-------------|-----------|
| [Z/OS et TSO/ISPF](zos-tso/) | Système z/OS, TSO, ISPF | 5 chapitres |
| [JCL](jcl/) | Job Control Language | 4 chapitres + TP |
| [VSAM](vsam/) | Virtual Storage Access Method | 7 chapitres |
| [Algorithmique](algorithmique/) | Algorithmique et programmation structurée | 9 chapitres |
| [COBOL](cobol/) | Langage COBOL et programmation mainframe | 12 chapitres |
| [CICS](cics/) | Moniteur transactionnel CICS | 8 chapitres |
| [DB2/SQL](db2/) | Bases de données relationnelles et SQL | 10 chapitres |

## Parcours recommandé

```
1. Module Z/OS et TSO/ISPF (fondamentaux)
   └── Chapitres I à V

2. Module JCL (contrôle des travaux)
   └── Chapitres I à IV + TP

3. Module VSAM (fichiers indexés)
   └── Chapitres I à VII

4. Module Algorithmique (logique de programmation)
   └── Chapitres I à VII (9 fiches)

5. Module COBOL (programmation)
   └── Chapitres I à XII

6. Module CICS (transactionnel)
   └── Chapitres I à VIII

7. Module DB2/SQL (bases de données)
   └── Chapitres I à X
```

## Prérequis par module

### Z/OS et TSO/ISPF
- Aucun prérequis technique
- Curiosité pour les systèmes d'information d'entreprise

### JCL
- Module Z/OS et TSO/ISPF recommandé
- Compréhension de l'environnement mainframe

### VSAM
- Module JCL recommandé
- Connaissance des Data Sets z/OS

### Algorithmique
- Aucun prérequis technique
- Logique et raisonnement mathématique de base

### COBOL
- Module Z/OS et TSO/ISPF recommandé
- Module JCL recommandé
- Connaissance de base en programmation utile

### CICS
- Maîtrise du module COBOL
- Connaissance de l'environnement z/OS

### DB2/SQL
- Maîtrise du module COBOL
- Connaissance de l'environnement z/OS (ISPF, JCL)
- Module CICS recommandé pour l'Embedded SQL
