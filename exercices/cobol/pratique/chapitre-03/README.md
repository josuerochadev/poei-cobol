# Exercices COBOL - Chapitre III

## Thème : Déclaration des variables

Ce chapitre approfondit la déclaration et manipulation des variables COBOL.

## Fichiers

| Programme | Description |
|-----------|-------------|
| `C03-NIVEAU88.cbl` | Utilisation des niveaux 88 (conditions) |
| `C03-REDEFINES.cbl` | Clause REDEFINES pour redéfinir des zones |

## Compilation et exécution

```bash
# Compiler un programme
cobc -x C03-NIVEAU88.cbl -o C03-NIVEAU88

# Exécuter
./C03-NIVEAU88
```

## Concepts abordés

- Niveaux de données (01, 05, 88)
- Conditions avec niveau 88
- Clause REDEFINES
- Redéfinition de zones mémoire
