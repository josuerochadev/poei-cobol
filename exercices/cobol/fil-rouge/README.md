# Projet Fil Rouge - COBOL/JCL

## Description
Projet de mise en pratique COBOL/JCL - Gestion financière (clients, comptes, produits, régions).

## Structure des exercices

| Ex | Dossier | Fichiers | Description |
|----|---------|----------|-------------|
| 01 | ex01-fichiers-base | 4 JCL | Création fichiers VSAM de base (Client, Compte, Produit, Région) |
| 02 | ex02-referentiels | 3 JCL | Fichiers référentiels (Compétence, Fonction, Médecin) |
| 03 | ex03-credit-debit | 2 JCL | Extraction crédit/débit |
| 04 | ex04-regions | 4 JCL | Traitement par région |
| 05 | ex05-extraction | 2 JCL | Extraction CR/DEB |
| 06 | ex06-tri | 1 JCL | Tri simple |
| 07 | ex07-tri-avance | 1 JCL | Tri avancé |
| 08 | ex08-index-alternatif | 1 JCL | Index alternatif VSAM (AIX) |
| 09 | ex09-index-alternatif-2 | 1 JCL | Index alternatif VSAM (AIX) suite |
| 10 | ex10-editer-index-alternatif | 2 CBL + 4 JCL | Édition index alternatif (AIX) |
| 11 | ex11-merge | 1 JCL | Fusion de fichiers (MERGE) |
| 12 | ex12-omit | 1 JCL | Exclusion d'enregistrements (OMIT) |
| 13 | ex13-ajout | 1 CBL + 2 JCL | Ajout d'enregistrements |
| 14 | ex14-sous-programmes | 4 CBL + 2 JCL | Sous-programmes (MAIN, CPT, PRO, REG) |
