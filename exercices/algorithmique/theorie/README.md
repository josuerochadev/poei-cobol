## Utilisation

1. Lire le chapitre correspondant dans `cours/algorithmique/`
2. Répondre au QCM sans consulter le cours
3. Vérifier les réponses avec les solutions fournies
4. Score minimal recommandé : 18/25 (72%)

## Barème

| Score | Niveau |
|-------|--------|
| 23-25 | Excellent - Maîtrise complète |
| 18-22 | Bien - Quelques révisions mineures |
| 13-17 | Moyen - Relire le chapitre |
| < 13 | Insuffisant - Revoir le cours en détail |
