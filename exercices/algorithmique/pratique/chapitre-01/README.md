## Points Cles a Retenir

| Pattern | Initialisation | Operation |
|---------|---------------|-----------|
| Somme | S := 0 | S := S + valeur |
| Produit | P := 1 | P := P * valeur |
| Comptage | C := 0 | C := C + 1 |
| Minimum | min := +infini | Si x < min Alors min := x |
| Maximum | max := -infini | Si x > max Alors max := x |
