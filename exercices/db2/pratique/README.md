# Exercices Pratiques Complementaires - DB2/SQL

Ce dossier contient des exercices pratiques supplementaires pour approfondir certains concepts non couverts dans les TP de la formatrice.

## Contenu

| Fichier | Theme | Description |
|---------|-------|-------------|
| [exercices-ddl.sql](exercices-ddl.sql) | DDL | CREATE, ALTER, DROP TABLE/INDEX |
| [exercices-dml.sql](exercices-dml.sql) | DML | INSERT, UPDATE, DELETE |

## Prerequis

Ces exercices utilisent un schema different des TP pour pratiquer la creation de tables.

## Objectifs

1. **DDL** : Pratiquer la creation et modification de structures
2. **DML** : Pratiquer les operations de manipulation de donnees
3. **Transactions** : Comprendre COMMIT et ROLLBACK
