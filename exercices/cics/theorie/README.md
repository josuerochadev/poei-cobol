## Utilisation

1. Lire le chapitre correspondant dans `cours/cics/`
2. Répondre au QCM sans consulter le cours
3. Vérifier les réponses avec les solutions fournies
4. Score minimal recommandé : 72%

## Barème

| Score | Niveau |
|-------|--------|
| 90-100% | Excellent - Maîtrise complète |
| 72-89% | Bien - Quelques révisions mineures |
| 50-71% | Moyen - Relire le chapitre |
| < 50% | Insuffisant - Revoir le cours en détail |
